# STTail

StockTwits querier

## [UI Client (create-react-app)](./client/)

```bash
cd client
yarn install
yarn start
```

## [REST API Server (Express.js backend to handle CORS)](./server/)

```bash
cd server
echo "CORS_ALLOWED_HOSTS='http://localhost:3001'" > .env
yarn install
yarn start
```
